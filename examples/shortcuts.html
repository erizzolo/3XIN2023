Useful keyboard shortcuts for Visual Studio Code

Elenchi di scorciatoie da tastiera per Visual Studio Code

File operation

  • Open File: Ctrl + O Opens a dialog window to open a file in the editor...
  • Save: Ctrl + S Save the current editor file...

Text editing

  1. Undo: Ctrl + Z Undo last operation...
  2. Redo: Ctrl + Y Redo last operation... also Ctrl + Shift + Z
